## 고요신세벌(Goyo Sebeol) V1.0 vs V1.1 레이아웃 개선 분석 보고서

---

### 1. 서론: 3x5 그리드 환경의 배열 최적화 전략

고요신세벌 자판은 **3x5(30키)**라는 물리적 제약 내에서 한글 입력 효율을 극대화한 인체공학적 배열입니다. '날개셋' 입력기 오토마타를 활용하여 초·중·종성 3단계 로직을 그리드에 최적화했습니다.

- **v1.1 목적:** 타건 데이터 분석을 통한 **SFB(Same Finger Bigram, 동일 손가락 연타)** 원천 방어 및 **'골든존'** 점유율 확대.
    
- **핵심 전략:** 오토마타 단계와 물리적 배치의 일치, 타건 리듬감 확보.
    

---

### 2. 주요 변경 내용 상세 분석 (v1.0 → v1.1)

v1.1은 입력 단계 간 간섭을 최소화하고 **인롤링(Inward Rolling)** 체계를 재정립하는 데 집중했습니다.

|변경 항목|v1.0 배치 (등급/점수)|v1.1 배치 (등급/점수)|설계 의도 및 전략적 배경|
|---|---|---|---|
|**오른손 [P], [;] 추가**|해당 없음|**ㅗ, ㅜ** (C / 5점)|이중모음 확장(Stage 2, 3) 및 검지 확장열 활용|
|**왼손 중성 [ㅐ] ↔ [ㅔ]**|[F] ㅔ (S / 10), [V] ㅐ (C / 5)|**[F] ㅐ** (S / 10), **[V] ㅔ** (C / 5)|빈도 21위(ㅐ)를 최상위 골든존([F])에 배치|
|**왼손 종성 [ㅂ] ↔ [ㅎ]**|[D] ㅂ (S / 10), [A] ㅎ (C / 5)|**[D] ㅎ** (S / 10), **[A] ㅂ** (C / 5)|고빈도 중성 'ㅏ'([D])와 종성 'ㅂ' 간 SFB 회피|
|**왼손 종성 [ㅊ] ↔ [ㅆ]**|[Q] ㅊ (C / 5), [B] ㅆ (D / 3)|**[Q] ㅆ** (C / 5), **[B] ㅊ** (D / 3)|받침 빈도 6위(ㅆ)를 접근성 유리한 상단열로 이동|

### 3. 정량적 평가: 자소 빈도 기반 개선도

국립국어원 현대국어 빈도 데이터를 기준으로 산출한 수치적 개선 결과입니다.

- **중성 ㅐ/ㅔ 최적화:** ㅐ(21위)가 ㅔ(22위)보다 약 16,000회 더 자주 사용됨을 확인했습니다. ㅐ를 홈로우([F])로 격상하여 물리적 접근 효율을 **약 100% 상승**시켰습니다.
    
- **SFB 확률 감소:** 중성 1위 'ㅏ'([D])와 종성 'ㅂ'의 충돌을 해결했습니다. 종성 위치를 'ㅎ'으로 교체하여 해당 위치의 SFB 발생 빈도를 **약 83.1% 감소**(159,597회 → 26,948회)시켰습니다.
    
- **종성 부하 재분배:** 고빈도 종성 'ㅆ'(6위)을 상단열([Q])로 배치하여 인롤링 흐름을 강화하고 전체 손가락 부하를 평준화했습니다.
    

---

### 4. 타건 흐름(Flow) 및 인체공학적 효과

#### 4.1 롤링(Rolling) 메커니즘 정교화

- **인롤링 강화:** 쌍자음 입력 시 안쪽으로 밀어치는 흐름을 적용했습니다. (예: ㄲ=kj, ㅉ=lk, ㅃ=;l, ㄸ=iu, ㅆ=oi)
    
- **아웃롤링 전략 활용:** 저빈도 자소(ㅋ, ㅌ)에만 예외적으로 적용하여 인지 혼선을 방지했습니다.
    

#### 4.2 오른손 확장성 및 코드(Chord) 설계

- 오른손 [P], [;]에 보조 모음(ㅗ, ㅜ)을 배치하여 이중모음(ㅘ, ㅝ 등) 입력 시 별도 레이어 전환 없이 연속적인 타건이 가능하도록 설계했습니다.
    

#### 4.3 왼손 소지(Pinky) 부하 관리

- 검지(40%), 중지(39%), 약지(19%), **소지(2%)** 비중으로 설계하여 근육 피로도를 최소화하고 고속 타건 리듬을 유지합니다.
