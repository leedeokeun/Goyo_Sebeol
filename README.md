# Goyo Sebeol (고요신세벌) - 3x5 Split Ortholinear Layout
[![License: CC BY-NC-SA 4.0](https://img.shields.io/badge/License-CC%20BY--NC--SA%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc-sa/4.0/)

3x5 그리드(한 손당 15키)라는 극한의 물리적 제약 내에서 타건 효율과 인체공학적 이점을 극대화한 **고요 신세벌 자판(Final 2026-02-14)**의 설계 원리와 조작법을 다룹니다. 본 자판은 날개셋 오토마타의 5단계 로직을 통해 복합 음절까지 매끄럽게 처리하며, 사용자 피로도를 최소화하는 '골든존' 설계를 채택하고 있습니다.

---

## 1. 고요 신세벌 레이아웃 구성


고요 신세벌은 손가락의 자연스러운 위치와 가동 범위를 고려하여 초성(오른손)과 중성/종성(왼손)을 분리 배치했습니다. 특히 고빈도 자소를 검지와 중지의 기본 위치인 '골든존(Tier 1)'에 배치하여 이동 거리를 최소화했습니다.

### 📊 고요 신세벌 최종 배열표 (QWERTY 대응)

| 손 | 구분 | Col 1 (소지) | Col 2 (약지) | Col 3 (중지) | Col 4 (검지) | Col 5 (검지확장) |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **왼손**<br>(중성/종성) | **상단** | Q: ㅒ / ㅊ | W: ㅓ / ㄷ | E: ㅣ / ㅌ | R: ㅛ / ㅇ | X: ㅑ / ㄱ |
| | **기본** | A: ㅕ / ㅎ | S: ㅡ / ㅅ | D: ㅏ / ㅂ | F: ㅔ / ㄴ | G: ㅢ / ㄹ |
| | **하단** | Z: ㅖ / ㅋ | X: ㅜ / ㅍ | C: ㅗ / ㅈ | V: ㅐ / ㅁ | B: ㅠ / ㅆ |
| **오른손**<br>(초성) | **상단** | Y: ㄹ | U: ㄷ | I: ㅁ | O: ㅊ | P: ㅍ |
| | **기본** | H: ㄴ | J: ㅇ | K: ㄱ | L: ㅈ | ;: ㅂ |
| | **하단** | N: ㅅ | M: ㅎ | ,: , | .: . | /: / |

> [!info] 설계 분석: 손가락 부하 및 골든존 전략
> * **검지 빈도 최적화:** 통계 데이터(검지빈도분석)에 따르면, 본 배열에서 검지(Col 4)와 검지 확장열(Col 5)의 타건 비중은 약 **40~49%**에 달합니다. 이는 가장 가동성이 뛰어난 검지에 핵심 자소(ㄴ, ㄹ, ㅇ, ㄱ 등)를 집중시켜 전체적인 타건 속도를 상향 평준화했음을 의미합니다.
> * **골든존(Tier 1-2):** 중지 기본 위치인 **J(ㅇ)**와 검지 기본 위치인 **K(ㄱ)**를 중심으로 설계하여, 한국어 초성 빈도 1, 2위를 차지하는 자소들의 접근성을 극대화했습니다.

---

## 2. 5단계 입력 프로세스 (오토마타 로직)


고요 신세벌은 복잡한 받침과 이중 모음을 처리하기 위해 기존 3단계를 넘어선 **5단계(Stage) 오토마타 로직**을 적용합니다. 이는 특히 '왉'과 같은 복합 음절에서 물리적 꼬임 없는 리듬을 제공합니다.

### 🔄 입력 단계 (Automata Flow)
* **[Stage 1: 초성]** 오른손으로 첫 자음을 입력합니다.
* **[Stage 2: 보조]** (선택) 오른손에서 'ㅗ/ㅜ'를 입력하거나 쌍자음 콤보를 수행합니다.
* **[Stage 3: 중성]** 왼손으로 모음을 입력합니다.
* **[Stage 4: 종성 1]** 왼손으로 첫 번째 받침을 입력합니다.
* **[Stage 5: 종성 2]** (선택) 복합 받침의 두 번째 자음을 입력합니다.

> **💡 복합 음절 '왉' 입력 예시**
> 1. `ㅇ` (오른손 J) - *Stage 1*
> 2. `ㅗ` (오른손 H 확장 입력) - *Stage 2*
> 3. `ㅏ` (왼손 D) - *Stage 3*
> 4. `ㄹ` (왼손 G) - *Stage 4*
> 5. `ㄱ` (왼손 X 확장 입력) - *Stage 5*
> 
> *이처럼 오른손(초성/보조)에서 왼손(중성/종성)으로 이어지는 일방향 흐름은 타건 리듬을 유지하며 인지적 부하를 줄여줍니다.*

---

## 3. 특수 입력: 쌍자음 및 격음 콤보 (Chord)


3x5 배열의 공간 제약을 극복하기 위해, 고요 신세벌은 손가락의 굴림 방향을 활용한 **롤링 콤보(Rolling Chord)** 방식을 채택했습니다.

### 🔗 주요 콤보 및 타건 방향

| 구분 | 대상 자소 | 입력 조합 (QWERTY) | 타건 원리 (Ulnar to Radial) |
| :--- | :--- | :--- | :--- |
| **인롤링 (In-roll)** | ㄲ, ㅉ, ㅃ, ㄸ, ㅆ | `kj`(ㄲ), `lk`(ㅉ), `;l`(ㅃ), `iu`(ㄸ), `oi`(ㅆ) | 소지/약지 → 검지 방향으로 안쪽 굴림 |
| **아웃롤링 (Out-roll)** | ㅋ, ㅌ | `jk`(ㅋ), `ui`(ㅌ) | 검지 → 약지 방향으로 바깥쪽 굴림 |
| **종성 ㄲ** | ㄲ | `df` 또는 `fd` | 왼손 검지/중지 제자리 연타 또는 교차 |

> [!abstract] 인지적 직관성 분석
> 인체 구조상 소지에서 검지 방향으로 흐르는 **'인워드 롤링(Inward Rolling)'**은 가장 빠르고 자연스러운 동작입니다. 이를 고빈도 쌍자음에 배정하고, 강한 타격감이 느껴지는 **'아웃워드 롤링(Outward Rolling)'**을 격음에 배정하여 기능적 직관성을 확보했습니다.

---

## 4. 실전 타건 최적화 가이드

* **검지 확장열의 효율적 활용:** 오른손 'ㅗ(H)', 'ㅜ(Y)' 입력 시, 약지를 뻗는 대신 검지를 안쪽으로 확장하여 타건하십시오. 이는 손목의 회전을 최소화하고 타건 안정성을 높입니다.
* **제자리 연타(SFB) 제로화 설계:** 왼손은 고빈도 결합인 'ㅏ(D) + ㄴ(F)', 'ㅓ(W) + ㅇ(R)' 등이 서로 다른 손가락에 배정되어 있습니다. 특히 왼손 중성에서 종성으로 이어지는 흐름은 대부분 인워드 롤링이 발생하도록 설계되어 연속 타건 시의 피로도를 획기적으로 낮췄습니다.
* **오른손 초성 전담의 이점:** 오른손은 초성 위주로 배치되어 있어 러닝커브가 매우 낮습니다. 사용자는 왼손의 중성-종성 연결 흐름만 익히면 3x5 극한의 공간에서도 전문적인 고속 타건이 가능합니다.
